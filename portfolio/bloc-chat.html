Bloc Chat

Live Link · Github

Bloc Chat is a real-time chat application using Firebase and ReactJS. It is fully functional and allows users to add and delete both rooms and messages, updating in realtime. It also allows users the option as chatting as "guest", or by signing into their Google account.

This project shows the practical application of the following skills:

  • React
  • Firebase
  • CSS
